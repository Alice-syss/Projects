# Number Guessing Game

A simple command-line game built with Python where you try to guess a randomly chosen number between 1 and 100.

## How to Run

1. Make sure Python is installed
2. Run this command in your terminal:
   python guess.py

## Features
- Random number generation
- Tracks number of attempts
- Input validation
- Play again option
